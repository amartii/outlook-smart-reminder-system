# > Outlook Smart Reminder System

**Accenture Internal Tool** — Agente de seguimiento inteligente de emails para Microsoft Outlook / Office 365.

Conecta directamente con tu cuenta de Outlook mediante **OAuth2** (Microsoft Graph API). Asigna categorías a los contactos (Blue, Manager, Senior Manager, VIP), envía recordatorios automáticos cuando no hay respuesta en el plazo configurado, y detecta etiquetas en las respuestas para reprogramar recordatorios sin intervención humana. Todo corre en local.
