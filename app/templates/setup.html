¿Por qué necesito un Client ID?
Microsoft eliminó la autenticación con usuario/contraseña en 2022. La única forma de acceder a Outlook es con OAuth2. Registrar la app es gratis con cualquier cuenta Microsoft y tarda 3 minutos. Ver instrucciones ▾
  1. Abre portal.azure.com e inicia sesión con cualquier cuenta Microsoft (puede ser @outlook.com personal)
  2. Busca "App registrations" → New registration
  3. Name: Outlook Smart Reminder · Supported account types: "Accounts in any organizational directory and personal Microsoft accounts"
  4. Click Register → copia el Application (client) ID
  5. Ve a API permissions → Add → Microsoft Graph → Delegated: Mail.Send, Mail.ReadWrite, Contacts.Read, User.Read, offline_access
  6. Ve a Authentication → activa "Allow public client flows" → Save
  7. Pega el Client ID abajo y pulsa Iniciar sesión
