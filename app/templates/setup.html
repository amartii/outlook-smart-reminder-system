Guía rápida — Registro en Azure AD
  1. Ve a portal.azure.com → Azure Active Directory → App registrations → New registration
  2. Name: Outlook Smart Reminder — Supported account types: Single tenant
  3. Copia Application (client) ID y Directory (tenant) ID
  4. Certificates & secrets → New client secret → copia el valor (solo visible una vez)
  5. API permissions → Add → Microsoft Graph → Application permissions:
    Mail.Send · Mail.Read · Contacts.Read · User.Read.All
  6. Haz clic en Grant admin consent
